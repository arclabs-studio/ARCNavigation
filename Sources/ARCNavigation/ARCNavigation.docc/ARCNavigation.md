# ``ARCNavigation``

A type-safe, testable SwiftUI navigation system for modern iOS apps.

## Overview

ARCNavigation provides a powerful yet simple navigation solution for SwiftUI applications. Built on top of SwiftUI's `NavigationStack`, it offers type-safe routing, comprehensive testing support, and a clean API that scales from simple to complex navigation flows.

### Key Features

- **Type-Safe Navigation**: Enum-based routes with compile-time safety
- **Centralized Management**: Single source of truth for navigation state
- **Fully Testable**: Unit test navigation flows without UI
- **Clean API**: Simple, intuitive methods for all navigation operations
- **Zero Dependencies**: Built entirely on SwiftUI and Swift standard library
- **Swift 6 Ready**: Full strict concurrency support
